## 一、需求

自动抓取网络资源的软件

（1）资源：网页、图片、音乐、视频等

（2）自动抓取：一旦开始运行，就不再需要其它过多的操作

## 二、总体设计

![image-20200613103658160](/home/wch/.config/Typora/typora-user-images/image-20200613103658160.png)

总体的设计流程是：

（1）首先得到爬取的种子(URL)

（2）根据爬取种子下载资源（页面）

（3）解析页面，然后提取更多的URL信息

（4）对页面进行持久化操作

（5）根据提取的URL再进行下载操作

（6）重复（2）-（5）
